# Revenue Aggregator

This is a  React application built to aggregate product revenue data across multiple branches, featuring real-time filtering, sorting, and dynamic total revenue calculation.

## 🚀 Live Demo

## 🛠 Features
- **Data Aggregation:** Efficiently merges products from various branch sources and calculates totals.
- **Search & Filter:** Real-time, case-insensitive product filtering.
- **Sorting:** Alphabetical organization of the product list.
- **Automated Testing:** Comprehensive unit tests implemented using **Jest** and **React Testing Library**.

## 💻 Technical Stack
- **Framework:** React 19 (Vite)
- **Testing:** Jest, @testing-library/react, @testing-library/jest-dom
- **Transpilation:** Babel (configured for JSX support in Jest)
- **Language:** JavaScript (ES Modules)
- **Styling:** CSS3 (Responsive Design)
